# mygrep

`mygrep` — это утилита фильтрации текстового потока, аналог команды UNIX `grep`.  
Программа читает текстовый файл или STDIN и выводит строки, соответствующие заданному шаблону.

## Возможности

Поддерживаются следующие флаги:

- `-A N` — выводит N строк после совпадения (контекст "after").  
- `-B N` — выводит N строк перед совпадением (контекст "before").  
- `-C N` — выводит N строк до и после совпадения (эквивалент `-A N -B N`).  
- `-c` — выводит только количество совпадений.  
- `-i` — игнорирует регистр.  
- `-v` — инвертирует фильтр (выводит строки, не содержащие шаблон).  
- `-F` — шаблон рассматривается как точная строка, а не регулярное выражение.  
- `-n` — выводит номера строк перед текстом.

## Установка и сборка

`make build` - Соберите бинарник
`make clean` - Очистка сборки
`make test` - Тесты

Поддерживаются сочетания флагов, например:  
```bash
./mygrep -i -v -n -C 2 "pattern" file.txt
